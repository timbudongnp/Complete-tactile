# Complete Tactile Installation

Coming Soon landing page for **Complete Tactile Installation** — a tactile indicator and accessibility compliance installation company based in Greater Sydney, NSW, with 15+ years of industry experience.

## Live Site

[https://www.completetactile.com.au](https://www.completetactile.com.au)

## Features

- Responsive landing page (mobile, tablet, desktop)
- Gold/black professional color scheme
- Quote request form powered by [Web3Forms](https://web3forms.com)
- Client-side form validation with error handling
- Contact confirmation modals for CTA buttons
- Security headers (CSP, X-Frame-Options, Referrer-Policy)
- SEO meta tags and Open Graph support
- Accessibility compliant (WCAG AA, prefers-reduced-motion, focus states)

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- Google Fonts (Poppins)
- Web3Forms (form backend)
- No frameworks or build tools required

## Deployment

Static site hosted via GitHub Pages. No build step required.
